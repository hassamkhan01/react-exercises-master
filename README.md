# Getting Started with Create React App

This project was created with [Create React App](https://github.com/facebook/create-react-app). The pre-requisites for this project are Node.JS (v19.x.x) and Node package manager npm (v9.x.x).

## Running the project

Open Terminal (make sure that Node.js is properly setup) and naviaget to the project director, e.g.:

### `cd C:\projects\react-exercices`

Install required dependencies with:

### `npm install`

Launches the runner in the interactive mode.

### `npm start`
